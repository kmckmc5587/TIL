## 2. 해당 폴더에서 Git 버전 관리 시작하기

```bash
$ git init
```

- 주의! (master)라고 되어 있으면 상위 폴더를 확인하자.
- 명령어를 입력하게 되면 .git 폴더가 생성된다.
